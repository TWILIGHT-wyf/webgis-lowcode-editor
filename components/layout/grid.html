vGrid 网格布局 ​

基于 CSS Grid 的布局容器组件。

基础用法 ​

vue
<script setup>
import { vGrid } from '@twi1i9ht/visual-lib'
</script>

<template>
  <vGrid :columns="3" :gap="16">
    <div v-for="i in 6" :key="i">Item {{ i }}</div>
  </vGrid>
</template>

Props ​

属性类型默认值说明
columnsnumber | string1列数或 grid-template-columns
rowsnumber | string—行数或 grid-template-rows
gapnumber | string0间距
rowGapnumber | string—行间距
columnGapnumber | string—列间距

插槽名说明
default网格子元素
